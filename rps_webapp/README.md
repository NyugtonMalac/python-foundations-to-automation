# Rock • Paper • Scissors — Flask Web App

## Overview

This project is a web-based adaptation of a two-player
Rock-Paper-Scissors console exercise.

The application was developed to extend a basic loop and
input-validation task into a fully deployable Flask web app.

## Features

- Two-player mode
- Odd-round validation (server-side)
- Session-based game state
- Controlled error handling
- Mobile-friendly UI
- Deployed in production environment

## Tech Stack

- Python
- Flask
- HTML (render_template_string)
- Virtual environment
- Passenger (production deployment)
